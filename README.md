# 🪙 CryptoDash - מערכת ניהול מטבעות קריפטו

מערכת מתקדמת למעקב אחר מטבעות קריפטו בזמן אמת, בנויה עם jQuery, Bootstrap והטמעת CoinCap API.

![Hebrew RTL](https://img.shields.io/badge/language-Hebrew%20RTL-blue)
![jQuery](https://img.shields.io/badge/jQuery-3.7.1-0769AD?logo=jquery)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3.3-7952B3?logo=bootstrap)

## ✨ תכונות עיקריות

### 📊 רשימת מטבעות
- **100 מטבעות מובילים** - טעינה אוטומטית של המטבעות המובילים בשוק
- **מידע מפורט** - דירוג, מחיר נוכחי, שינוי 24 שעות
- **הצגת פרטים נוספים** - Market Cap, Supply, ומידע נוסף בלחיצה
- **אנימציות חלקות** - כרטיסים עם אנימציית fade-in מדורגת

### 🔍 חיפוש מתקדם
- חיפוש בצד לקוח (client-side) - מהיר וללא עומס על השרת
- תמיכה בחיפוש לפי:
  - שם המטבע (Bitcoin, Ethereum...)
  - סמל (BTC, ETH...)
  - מזהה (bitcoin, ethereum...)
- חיפוש בזמן אמת תוך כדי הקלדה

### 💼 תיק מעקב (Watchlist)
- **עד 5 מטבעות** - שמירת מטבעות מועדפים
- **מעקב מתמיד** - שמירה ב-localStorage
- **ניהול קל** - הוספה והסרה בלחיצה אחת
- **החלפה חכמה** - מודל לבחירת מטבע להחלפה כשהתיק מלא

### 📈 גרף בזמן אמת
- **ריענון אוטומטי** - עדכון כל 30 שניות
- **תצוגה מרובת מטבעות** - גרף מקווקו לכל המטבעות בתיק
- **צבעים דינמיים** - כל מטבע בצבע ייחודי
- **CanvasJS** - גרף אינטראקטיבי ומהיר

### 🎨 חווית משתמש משופרת
- **עיצוב רספונסיבי** - תמיכה מלאה במובייל וטאבלט
- **RTL מלא** - תמיכה בעברית מימין לשמאל
- **התראות Toast** - הודעות ויזואליות לפעולות המשתמש
- **כפתור חזרה למעלה** - ניווט נוח בדף ארוך
- **אנימציות חלקות** - מעברים ואפקטי hover מלוטשים
- **Smooth scroll** - גלילה חלקה בין קטעים

### ⚡ ביצועים
- **Cache חכם** - שמירת נתונים ל-30 שניות למניעת קריאות מיותרות
- **טעינה אסינכרונית** - ממשק משתמש רספונסיבי
- **טיפול בשגיאות** - הודעות ברורות ואופציות לטעינה מחדש

## 🚀 התחלה מהירה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- חיבור לאינטרנט (לטעינת ספריות CDN ו-API)

### הפעלה מקומית

1. **Clone או הורדת הפרויקט**
   ```bash
   git clone <repository-url>
   cd cripto
   ```

2. **פתיחת הפרויקט**

   פשוט פתח את הקובץ `index.html` בדפדפן, או השתמש בשרת מקומי:

   ```bash
   # Python 3
   python -m http.server 8000

   # Python 2
   python -m SimpleHTTPServer 8000

   # Node.js (npx http-server)
   npx http-server
   ```

3. **גלוש ל:**
   ```
   http://localhost:8000
   ```

## 🔌 API

הפרויקט משתמש ב-[CoinCap API](https://coincap.io/) לקבלת נתוני קריפטו:

- **Endpoint:** `https://api.coincap.io/v2/assets`
- **Rate Limit:** מוגבל (לכן משתמשים ב-cache)
- **Authentication:** API Key מוגדר (אופציונלי)

## 🐛 טיפול בבעיות

### הנתונים לא נטענים
- בדוק חיבור אינטרנט
- פתח Developer Console (F12) לבדיקת שגיאות
- נסה לרענן את הדף (F5)

- וודא שיש מטבעות בתיק המעקב
- המתן 30 שניות לעדכון ראשון
- בדוק ש-CanvasJS נטען בהצלחה

### בעיות Cache
- נקה localStorage בדפדפן
- נקה Cache של הדפדפן
- השתמש במצב פרטי (Incognito)
