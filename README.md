# פרויקט jQuery AJAX - מערכת מעקב קריפטו

## תיאור הפרויקט
מערכת לצפייה ומעקב אחר מטבעות קריפטו בזמן אמת.
הפרויקט בנוי בטכנולוגיות jQuery ו-Bootstrap עם שימוש ב-API חיצוני לקבלת נתונים.

## טכנולוגיות
- HTML5
- CSS3
- JavaScript
- jQuery 3.7.1
- Bootstrap 5.3.3 (RTL)
- CanvasJS (לגרפים)
- CoinGecko API

## תכונות
1. תצוגת 100 המטבעות המובילים
2. חיפוש לפי שם/סמל
3. תיק מעקב עד 5 מטבעות
4. גרף בזמן אמת עם רענון כל 30 שניות
5. שמירה ב-localStorage

## מבנה הקבצים
```
cripto/
├── index.html       - דף HTML ראשי
├── css/
│   └── style.css    - עיצוב
└── js/
    ├── app.js       - לוגיקה ראשית
    └── accessibility.js - נגישות
```

## הפעלה
1. פתיחת index.html בדפדפן
2. או הרצת שרת מקומי: `python -m http.server 8000`

## API
שימוש ב-CoinGecko API לקבלת נתוני מטבעות בזמן אמת.

## נגישות
הפרויקט כולל תמיכה בנגישות בסיסית.
